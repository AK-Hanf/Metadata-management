# Analytical Data Templates & Metadata Management

This folder contains standardized Excel templates for various analytical techniques commonly used in materials science and chemistry research, along with utilities for data conversion and metadata insertion.

## Contents

### Analytical Technique Templates
A comprehensive collection of 30+ Excel templates for different characterization methods:

- **Spectroscopy**: XPS, UV-Vis, IR, Raman, NMR, EA
- **Chromatography**: HPLC, GC, SEC
- **X-ray Techniques**: PXRD (Powder X-ray Diffraction), SXRD (Single Crystal X-ray Diffraction), XAS
- **Microscopy**: SEM (Scanning Electron Microscopy), TEM (Transmission Electron Microscopy)
- **Electrochemistry**: CV (Cyclic Voltammetry), CO, EDX
- **Thermal Analysis**: TG (Thermogravimetry), TPR (Temperature Programmed Reduction), TPO (Temperature Programmed Oxidation)
- **Materials Characterization**: BET, DLS (Dynamic Light Scattering), ICP, Conductivity, DRIFT, EIMS, ESIMS, PL (Photoluminescence), PL LT

### Data Files
- **20250515_cat synthesis.xlsx** - Synthesis data for catalyst testing (May 15, 2025)
- **20250515_cat test and analytics.xlsx** - Analytical results and testing data for catalysts (May 15, 2025)

### Utilities
- **txt to JSON converter** - Tool for converting text format data to JSON format

### Metadata
- **Metadata** - Placeholder/template for manual metadata insertion

## Purpose

These templates are designed to:
1. Standardize data collection across different analytical techniques
2. Facilitate consistent metadata insertion for scientific datasets
3. Enable seamless conversion of experimental data to JSON format for database integration and analysis
4. Support research workflows in materials characterization and catalysis studies

## Usage

1. Select the appropriate Excel template for your analytical technique
2. Fill in experimental data following the template structure
3. Insert relevant metadata (sample information, conditions, etc.)
4. Use the provided conversion utilities to convert data to JSON format if needed
